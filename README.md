# 项目简介

这是一个使用Vite+React构建的网站，是 [喵和の女仆咖啡 miaohe.cafe](https://miaohe.cafe) 的网站源代码。

# 构建方式

克隆项目
```
git clone https://github.com/VanVodkaer/MiaoHeCafe.git
```

安装依赖
```
cd MiaoHeCafe
npm i
```

构建项目
```
npm run build
```
